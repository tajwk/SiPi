SiPi Quick Start Guide

Welcome to SiPi, your browser-based interface for Sidereal Technology telescope control. This manual will help you set up, connect, and start operating your telescope quickly and confidently.

1. System Setup

  1. Power on your SiPi by plugging in the USB power cable
  2. On your device (tablet, or phone, or PC), connect to the Pi's Wi-Fi network.
    SSID and password can be configured in the settings if needed. Default SSID=SiPi password=12345678
  3. Open your browser and go to:
    https://192.168.11.1:5000
If the page does not load, confirm that you are on the correct Wi-Fi and the Pi is fully booted.

2. Settings

  • Set your site location by entering your current SiteLongitude and SiteLatitude
  • Set your Mount Type and Scope Encoder Settings
  • Enabled/Disable Drag mode and Az/RA No Wrap
  • Click the SiTechExe Config button to show/hide the SiTechExe options.
  • You can backup your SiTechExe configuration or restore the backed up configuration
    • Always click Save Config after making changes to SiTechExe!
  • System Time can be set by entering a time HH:MM and Clicking Set. Clicking Sync will enter your device time.
  • Set your Wi-Fi SSID and password if desired.
  • Hanpad tilt and vibration settings can be set here. (Note these features are not supported by all web browsers)
  • Check for and apply SiPi updates

3. Buttons

  • Sky will switch to SkyView
  • Auto is used for normal operation. Manual disengages the motors for maintenance, adjustment or after parking
  • Park or Unpark your mount. Set the park position in Settings
  • Click the ⚙ button to access SiPi and controller settings
  • Click the ⛶ button to go fullscreen. This is very helpful in Night Mode.
  • Click the Help button to access this help file

4. Main Interface Overview

  • Status Panel: Live RA/Dec, Alt/Az, Controller Status, Time, LST, and HA
  • Object Search: Search for stars, Messier objects, planets, and more.
  • Model: Create a mount model to improve pointing and tracking
  • Handpad Controls: Slew the telescope with on-screen arrows.
  • Sidebar: Mount Control, Settings and Help

5. Handpad Control

Use the on-screen handpad to:
  • Up/Down: Secondary axis (Declination/Altitude).
  • Left/Right: Primary axis (RA/Azimuth).
  • Change Speed: Tap the center to toggle Slew, Pan, or Guide modes.
  • START begins Tracking if the mount has been initialized.STOP Halts all motion instantly.

6. Sync on an Object

  1. Slew to a known object and center it in the eyepiece
  2. Enter the object name (e.g., Vega, M13) in the search bar.
  3. Click Search. Select your desired object from the results if prompted.
  4. Then Click Sync
  5. The mount status should now indicated that it's Tracking

7. Model Building

A mount model can greatly improve your pointing and tracking accuracy.
  1. Search for an object and perform a GoTo
  2. Use the handpad to center the object in the eyepiece and then press +Cal Pt.
  3. Repeat with additional objects across the sky.
  4. When done, click Save to save your pointing model.
  5. To create a new model, click Clear to remove all calibration points
  6. You can remove the last added Cal Pt by clicking - Cal Pt
You will start to see improvements after a few objects are added to the model. Cal Pts can be viewed & enabled/disabled in SkyView
If the mount remains stationary between uses, the model can be reused. If the mount is changes location, a new model is needed.

8. SkyView Planetarium

  1. Click Sky to open a live sky map
  2. The Obj menu has checkboxes that allow you to enable/disable each type
  3. Pan/zoom to explore, select (click) objects for info
  4. A selected object's coordinates will appear in the box below the SkyView projection. Clicking Sync will align the telescope with the selected object. Clicking Go will perform a GoTo the selected object.
  5. Search for objects (Coming Soon...)
  6. Click the refresh button for default zoom or the home button to return to the main screen

9. Typical Workflow

  1. Power on your Pi and telescope.
  2. Connect to the Pi's Wi-Fi and open SiPi.
  3. Search for and slew to a bright star.
  4. Center, then Sync.
  5. Create and Save Mount Model.
  6. Ready for GoTos all night!

10. Troubleshooting

  • Page won’t load? Check Wi-Fi and Pi power.
  • No telescope response? Make sure the SiTechEXE service is running (Settings)
  • Slow or unresponsive UI? Wait a minute after boot for initialization.

Encoder Settings

  • Ignore: Use this option if your mount does not have secondary encoders
  • Precise: Position correction will appear as soon as there is a disagreement between the motor and secondary encoders
  • Polite: Position correction will appear only during slewing. Corrections will not apply while tracking
  • Cascade: High position correction, but requires high resolution (10M+ Ticks per Rev)
